# Microsoft Intune Compliance Policy Lab

## 📌 Description

This project demonstrates the configuration of mobile device compliance policies using Microsoft Intune within a Microsoft 365 tenant environment.

The objective of this lab was to simulate a real-world enterprise mobile device management (MDM) setup by creating users, security groups, and compliance policies to enforce security standards on managed devices.

The environment was built using a Microsoft 365 tenant with Microsoft Intune enabled. Administrative tasks were performed through the Microsoft 365 Admin Center and Microsoft Intune Admin Center.

(In this demo I have used placeholder identities to protect real emails).

### 4. Android Compliance Policy Deployment

Created and configured an Android compliance policy with enterprise-level security requirements.

Configured compliance settings included:

* Block rooted devices
* Require Google Play Services
* Require up-to-date security provider
* Set device threat level to High
* Configure Microsoft Defender risk level compliance

These settings simulate typical corporate mobile security standards.
